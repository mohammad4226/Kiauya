# ديواني - Diwani

موقع ديوان الشعر والاقتباسات - **A Poetry & Quotes Website**

## 📝 الوصف (Description)

ديواني هو موقع تفاعلي يمكنك من خلاله:
- 📖 عرض مجموعة من الشعر والاقتباسات الملهمة
- ➕ إضافة شعر أو اقتباسات جديدة
- 🔍 البحث عن الشعر والاقتباسات
- 🏷️ تصفية حسب نوع المحتوى (شعر/اقتباسات)
- ✏️ تعديل المحتوى
- 🗑️ حذف المحتوى

## 🛠️ التقنيات المستخدمة (Technologies)

- **HTML5** - هيكل الصفحة
- **CSS3** - التصميم والمظهر
- **JavaScript (Vanilla)** - الوظائف التفاعلية

## 🚀 كيفية الاستخدام (How to Use)

1. **فتح الموقع**: قم بفتح ملف `index.html` في المتصفح
2. **عرض المحتوى**: يمكنك مشاهدة الشعر والاقتباسات المتوفرة
3. **البحث**: استخدم شريط البحث للبحث عن محتوى معين
4. **التصفية**: استخدم أزرار التصفية (الكل / الشعر / الاقتباسات)
5. **إضافة**: اضغط على زر "+" لإضافة شعر أو اقتباس جديد
6. **التعديل/الحذف**: استخدم أزرار التعديل والحذف على كل عنصر
